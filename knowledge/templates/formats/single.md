# 単独投稿テンプレート

単一のツイートで完結する投稿フォーマット。

## 制約

- 文字数: 140-280文字
- ハッシュタグ: 2-3個（末尾推奨）
- リンク: 任意（1つまで）
- 画像/動画: 任意

## 構成パターン

### パターン1: 問題提起型

```
[問題/課題の提示]

[解決策/気づきの提示]

[ハッシュタグ]
```

例:
```
SNSマーケティングで最も見落とされがちなのは「投稿タイミング」です。

データによると、平日12時台と21時台のエンゲージメントが最も高い。まずはこの時間帯を狙ってみましょう。

#SNSマーケティング #運用術
```

### パターン2: Tips型

```
【〇〇のコツ】

[箇条書きまたは簡潔な説明]

[ハッシュタグ]
```

例:
```
【Xフォロワーを増やす3つの基本】

1. 毎日同じ時間に投稿
2. リプライには必ず返信
3. 価値ある情報を惜しみなく共有

継続が何より大切です。

#X運用 #フォロワー増加
```

### パターン3: 気づき共有型

```
[気づき/学びの共有]

[具体的なエピソードや数字]

[ハッシュタグ]
```

例:
```
1年間Xを運用して気づいたこと。

「バズる投稿」を狙うより「継続できる投稿」を続ける方が、結果的にフォロワーが増える。

毎日1投稿を365日続けた結果、フォロワー0→5000人に。

#継続は力なり #X運用
```

## チェックリスト

- [ ] 文字数は140-280文字か
- [ ] 主張が明確か
- [ ] ハッシュタグは適切か（2-3個）
- [ ] 読みやすい改行が入っているか
- [ ] CTAが含まれているか（任意）
