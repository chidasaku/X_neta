# スレッドテンプレート

複数のツイートで構成されるスレッド形式の投稿フォーマット。

## 制約

- 各ツイート: 140-280文字
- ツイート数: 3-10投稿（5-7推奨）
- ハッシュタグ: 最終ツイートに2-3個
- 番号付け: 「1/5」「2/5」形式を推奨

## 構成

### 1. フック（1ツイート目）

興味を引く導入。以下のパターンが効果的:

- 「〇〇について、多くの人が誤解していることがあります」
- 「この3年で学んだ〇〇の真実をシェアします」
- 「〇〇で成功する人としない人の違いは、たった1つ」
- 「〇〇を知らないと、あなたは損をしています」

### 2. 本論（2-N-1ツイート目）

- 1ツイート1トピック
- 具体例・数字を含める
- 読者が次を読みたくなる構成に

### 3. まとめ（最終ツイート）

- 要点の再確認
- CTA（行動喚起）
- ハッシュタグ

## テンプレート例

```
🧵 1/5
SNSマーケティングで結果を出すために、絶対に押さえるべき5つのポイントをまとめました。

これを知らないと、どれだけ投稿しても成果が出ません。
```

```
2/5 【ポイント1: ターゲット設定】

「みんなに届けたい」は「誰にも届かない」

理想の顧客を1人決めて、その人に向けて書く。
年齢、職業、悩み、願望を具体的にイメージしましょう。
```

```
3/5 【ポイント2: 投稿時間の最適化】

ターゲットがSNSを見る時間帯に投稿する。

ビジネスパーソン向け→朝7時、昼12時、夜21時
主婦向け→朝10時、昼14時、夜22時

データを見て調整を。
```

```
4/5 【ポイント3: 価値提供ファースト】

「売り込み」より「役立つ情報」を先に。

信頼関係ができてから、初めてセールスが機能する。
80%価値提供、20%プロモーションが黄金比。
```

```
5/5 【まとめ】

1. ターゲットを1人に絞る
2. 投稿時間を最適化
3. 価値提供を先に

この3つを意識するだけで、反応は確実に変わります。

気になった方はフォローして続きもチェック！

#SNSマーケティング #運用術 #マーケティング
```

## チェックリスト

- [ ] フックで興味を引けているか
- [ ] 各ツイートが自己完結しているか
- [ ] 番号付けが正しいか
- [ ] 最終ツイートにCTAがあるか
- [ ] ハッシュタグは最終ツイートにあるか
