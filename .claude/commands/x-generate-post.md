# X Generate Post - 投稿生成コマンド

ナレッジベースの情報を元に、X投稿案を自動生成します。

## 実行フロー

### Step 1: トピック分析

ユーザー指定のトピックを分析:
- 関連するナレッジを検索
- コンテキスト構築

### Step 2: スタイル選択

投稿スタイルを確認:
- **professional**: ビジネス・専門的
- **casual**: カジュアル・親しみやすい
- **educational**: 教育的・解説

### Step 3: フォーマット選択

投稿フォーマットを確認:
- **single**: 単独投稿（1ツイート）
- **thread**: スレッド形式（複数ツイート）
- **quote**: 引用形式
- **tips**: Tips形式

### Step 4: 投稿生成

関連ナレッジとテンプレートを使用してClaude APIで生成。

### Step 5: 結果表示

```

フォーマット: single
参照ナレッジ: 3件

[生成された投稿テキスト]

[生成された投稿テキスト]

[生成された投稿テキスト]
文字数: 195
```

## オプション

| オプション | 説明 | デフォルト |
|-----------|------|-----------|
| --style | スタイル | professional |
| --format | フォーマット | single |
| --count | 生成数 | 3 |
| --knowledge | 参照ナレッジID | 自動検索 |

## 使用例

```bash
# 基本生成
/x-generate-post SNSマーケティングのコツ

# スタイル指定
/x-generate-post AI活用事例 --style casual

# スレッド形式
/x-generate-post 生産性向上の秘訣 --format thread

# 生成数指定
/x-generate-post 業界トレンド --count 5

# 特定ナレッジ参照
/x-generate-post --knowledge abc123
```

## 投稿テンプレート

### single（単独投稿）
- 文字数: 140-280文字
- ハッシュタグ: 2-3個
- CTA: 任意

### thread（スレッド）
1. フック（興味を引く導入）
2. 本論（2-5ツイート）
3. まとめ（CTA）

### tips（Tips形式）
- 番号付きリスト
- 簡潔なポイント
- 実践的な内容

## 品質チェック

生成された投稿は以下をチェック:
- ✅ 文字数（140-280文字）
- ✅ 重複確認
- ✅ ハッシュタグ形式
- ✅ リンク形式

## 注意事項

- 生成結果は案であり、投稿前に内容を確認してください
- センシティブな内容が含まれる場合があります
- 著作権に注意してください
