# X Knowledge Search - ナレッジ検索コマンド

ナレッジベースをセマンティック検索し、関連する情報を取得します。

## 実行フロー

### Step 1: クエリ分析

ユーザーの検索クエリを分析:
- 意図の理解
- キーワード抽出
- 検索戦略決定

### Step 2: 検索実行

1. クエリをベクトル化（OpenAI埋め込み）
2. 類似度計算（コサイン類似度）
3. Top-K結果取得

### Step 3: 結果表示

```
🔍 検索結果: "[クエリ]"

1. [タイトル1] (類似度: 0.92)
   カテゴリ: marketing | タグ: #SNS #運用
   > 関連する本文の抜粋...

2. [タイトル2] (類似度: 0.87)
   カテゴリ: engagement | タグ: #エンゲージメント
   > 関連する本文の抜粋...

3. [タイトル3] (類似度: 0.85)
   ...
```

## オプション

| オプション | 説明 | デフォルト |
|-----------|------|-----------|
| --category | カテゴリフィルター | なし |
| --tags | タグフィルター | なし |
| --limit | 結果数 | 5 |
| --mode | semantic/keyword/hybrid | semantic |

## 使用例

```bash
# 基本検索
/x-knowledge-search エンゲージメントを高める方法

# カテゴリ指定
/x-knowledge-search ハッシュタグ戦略 --category marketing

# 結果数指定
/x-knowledge-search 投稿タイミング --limit 10

# ハイブリッド検索
/x-knowledge-search 分析指標 --mode hybrid
```

## 検索モード

### semantic（デフォルト）
- ベクトル類似度による検索
- 意味的に関連する情報を取得

### keyword
- キーワードマッチによる検索
- 完全一致・部分一致

### hybrid
- semantic + keyword の組み合わせ
- より精度の高い結果

## 注意事項

- OpenAI APIキーが必要です
- ナレッジが登録されていない場合は結果が0件になります
- 類似度スコアが0.7未満の結果は表示されません
